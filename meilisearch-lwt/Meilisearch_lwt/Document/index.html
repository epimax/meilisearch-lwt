Module Meilisearch_lwt.Document

Module for basic document management. Provides APIs to insert and delete documents. There is no format checking for these APIs; you provide Yojson.Safe.t formatted documents and send them to the Meilisearch instance.

Documents are treated as Yojson.Safe.t. This module helps you manage such data on your Meilisearch instances.

Shared types and functions

type document_error = [
  1. | Meilisearch_lwt__.Meilisearch_error.t
  2. | `NoSuchDocument of string
  3. | `NoSuchIndex of string
]

Errors that can occur in this module: non-existent document/index, and also network/system errors.

val add_document : Meilisearch_lwt__.Meilisearch_client.t -> index_uid:string -> ('a -> Yojson.Safe.t) -> 'a -> (int, document_error) Stdlib.result Lwt.t

add_document client ~index_uid f doc sends a single document doc to the index index_uid. f is your generated to_yojson function, needed to build the request body. Asynchronous task, returns a task id when successful.

get_document client ~index_uid ~document_uid gets a document using its ID (primary key). Immediate request (not queued as an asynchronous task in Meilisearch), returns the document when present on the instance.
